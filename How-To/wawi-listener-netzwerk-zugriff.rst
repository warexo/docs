Lokaler Netzwerk Zugriff WAWI Listener / Warexo Bridge
#################################################################

In der aktuellen Chrome-Version erscheint oben links ein neues Popup mit dem Hinweis „Zugriff auf lokales Netzwerk“.
Diese Anfrage muss durch Klick auf „Zulassen“ bestätigt werden - insbesondere für den Betrieb der Kassen ist dies zwingend erforderlich.

.. attention:: Ohne diese Freigabe kann keine Verbindung zum WAWIListener hergestellt werden.

Die erteilte Berechtigung wird dauerhaft in Chrome gespeichert und kann unter
„Einstellungen → Datenschutz und Sicherheit → Website-Einstellungen → Zugriff auf lokales Netzwerk“
eingesehen oder angepasst werden.
